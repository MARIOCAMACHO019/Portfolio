Migración de Infraestructura a Azure

En este proyecto personal de nivel avanzado, asumí el reto de modernizar una infraestructura heredada (on-premise) simulada, migrando servicios críticos a la nube de Microsoft Azure. Mi objetivo principal fue garantizar la alta disponibilidad y la seguridad desde el diseño inicial de la arquitectura.


Diseñé e implementé una arquitectura de red Hub-and-Spoke, estableciendo Peering entre VNets para segmentar el tráfico de gestión del tráfico de producción. Configuré Azure Active Directory (Entra ID) sincronizándolo con un dominio local mediante Azure AD Connect para permitir una identidad híbrida, facilitando el Single Sign-On (SSO) para los usuarios.


Para el almacenamiento, desplegué cuentas con redundancia geográfica (GRS) y políticas de ciclo de vida. Además, automaticé el despliegue de máquinas virtuales utilizando scripts de PowerShell y plantillas ARM, asegurando que cada VM cumpliera con las normativas de seguridad mediante Network Security Groups (NSG) altamente restrictivos.


Tecnologías aplicadas: Azure VNet, NSG, Azure AD Connect, PowerShell, Blob Storage.

Despliegue Corporativo con Windows Server

Lideré la implementación completa de un dominio de Active Directory para una empresa ficticia de 200 empleados con múltiples departamentos. El desafío consistía en crear una estructura lógica, segura y escalable desde cero utilizando Windows Server 2022.


Estructuré el directorio mediante una jerarquía de Unidades Organizativas (OU) granular. Implementé un esquema robusto de GPOs (Políticas de Grupo) para aplicar configuraciones de seguridad masivas: bloqueo de puertos USB, políticas de contraseñas complejas, despliegue de software automático y mapeo de unidades de red compartidas según el departamento.


Además, configuré los servicios de red esenciales: un servidor DNS con zonas directas e inversas y un servidor DHCP con ámbito tolerante a fallos (Failover) para garantizar que la red nunca perdiera conectividad ni asignación de IPs, incluso ante la caída de un controlador de dominio.


Tecnologías aplicadas: Windows Server 2022, AD DS, DNS, DHCP Failover, Group Policy Management.

Gestión y Optimización de BBDD Relacionales

Desarrollé y optimicé el backend de datos para un sistema de gestión de inventario. Mi trabajo se centró en el diseño de un esquema relacional eficiente en MySQL, priorizando la integridad de los datos y la velocidad de respuesta en consultas complejas.


Realicé la normalización de la base de datos hasta la tercera forma normal (3NF) para eliminar redundancias. Escribí procedimientos almacenados (Stored Procedures) y Triggers para automatizar tareas de mantenimiento y auditoría de cambios en los datos sensibles, asegurando que ninguna modificación pasara desapercibida.


Para mejorar el rendimiento, analicé los planes de ejecución de las queries más lentas e implementé índices estratégicos que redujeron significativamente el tiempo de respuesta. También configuré un sistema de copias de seguridad (Backups) automatizadas para asegurar la recuperación ante desastres.


Tecnologías aplicadas: MySQL Workbench, SQL Avanzado, Procedimientos Almacenados, Optimización de Índices.

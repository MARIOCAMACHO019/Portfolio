DESPLIEGUE WINDOWS SERVER

ADMINISTRACIÓN DE SISTEMAS & GPO

Descripción del Proyecto

Diseño de una infraestructura de dominio para 200 usuarios basada en Windows Server 2022. El foco principal fue la alta disponibilidad y la gestión centralizada.

Implementación Técnica

  • AD DS: Controladores de dominio redundantes.
  • Red: DNS interno y DHCP con Failover.
  • Seguridad: Políticas de Grupo (GPO) para hardening de clientes.
  • Automatización: Scripts PowerShell para gestión de usuarios.

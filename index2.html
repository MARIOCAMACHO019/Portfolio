DESPLIEGUE CORPORATIVO WINDOWS SERVER

ADMINISTRACIÓN DE SISTEMAS & DIRECTORIO ACTIVO

Descripción del Proyecto

Lideré la implementación completa de un dominio de Active Directory (AD DS) para una empresa ficticia de 200 empleados con múltiples departamentos. El desafío principal consistía en diseñar una estructura lógica, segura y escalable desde cero, utilizando Windows Server 2022 como base del sistema operativo.

El proyecto simulaba un entorno de producción real donde la continuidad del negocio era crítica, por lo que se implementaron medidas de redundancia en todos los servicios esenciales de la red.

Implementación Técnica (Nivel Avanzado)

  • Controladores de Dominio: Instalación y promoción de dos controladores de dominio (DC) configurados en alta disponibilidad (Primario y Secundario) para asegurar la replicación y evitar puntos únicos de fallo.
  • Estructura Lógica (OU): Diseño granular de Unidades Organizativas separando usuarios, equipos y servidores por departamento, facilitando la delegación de administración.
  • Servicios de Red Críticos:
    • DNS: Configuración de zonas de búsqueda directa e inversa y reenviadores condicionales.
    • DHCP: Implementación de ámbito con Failover (conmutación por error) para garantizar la asignación de IPs ininterrumpida.
  • Políticas de Grupo (GPO): Despliegue masivo de configuraciones de seguridad (bloqueo de USB, complejidad de contraseñas) y mapeo automático de unidades de red compartidas según el perfil del usuario.
  • Automatización: Creación de scripts en PowerShell para la importación masiva de usuarios desde archivos CSV, asignándoles automáticamente grupos y permisos.

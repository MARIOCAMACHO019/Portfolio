GESTIÓN Y OPTIMIZACIÓN DE BBDD

ADMINISTRACIÓN SQL & OPTIMIZACIÓN DE CONSULTAS

Descripción del Proyecto

Desarrollé y optimicé el backend de datos para un sistema ERP de gestión de inventario masivo. Mi trabajo se centró en el diseño de un esquema relacional eficiente utilizando MySQL Workbench, priorizando la integridad de los datos y la velocidad de respuesta en consultas complejas.

El objetivo era transformar una base de datos desestructurada en un sistema robusto capaz de soportar alta concurrencia y generar reportes en tiempo real sin degradar el rendimiento.

Implementación Técnica (Nivel Avanzado)

  • Modelado de Datos: Realicé la normalización de la base de datos hasta la 3ª Forma Normal (3NF) para eliminar redundancias y asegurar la integridad referencial.
  • Programación SQL Avanzada:
    • Triggers: Implementación de disparadores para auditar cambios en tablas sensibles (histórico de precios y modificaciones de stock).
    • Stored Procedures: Desarrollo de procedimientos almacenados para automatizar tareas recurrentes como el cierre de caja mensual y reportes de ventas.
  • Optimización de Rendimiento: Análisis de planes de ejecución (Query Execution Plans) para detectar cuellos de botella. Implementé índices estratégicos (B-Tree) que redujeron el tiempo de respuesta de las consultas más pesadas en un 70%.
  • Seguridad y Mantenimiento: Configuración de un sistema de copias de seguridad (Backups) lógicas automatizadas mediante mysqldump y tareas programadas (Cron Jobs) en entorno Linux.

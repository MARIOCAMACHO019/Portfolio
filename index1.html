INFRAESTRUCTURA HÍBRIDA AZURE

Descripción del Proyecto

En este proyecto, realicé una migración completa de servicios on-premise a la nube de Microsoft Azure. El objetivo era modernizar la infraestructura de una empresa simulada, asegurando alta disponibilidad y reduciendo costes de mantenimiento físico.

Implementación Técnica (Nivel Avanzado)

  • Redes Virtuales (VNet): Diseñé una arquitectura Hub-and-Spoke con segmentación de tráfico mediante subredes (Frontend, Backend, GatewaySubnet) y VNet Peering.
  • Seguridad Perimetral: Implementación de Network Security Groups (NSG) con reglas de entrada/salida estrictas, permitiendo solo tráfico necesario (puerto 443, 22 para gestión segura).
  • Conectividad Híbrida: Despliegue de VPN Gateway (Route-based) estableciendo un túnel IPsec Site-to-Site con la sede local.
  • Gestión de Identidad: Configuración de Azure Active Directory (Entra ID) sincronizado con el AD local para Single Sign-On (SSO).
